# MealApp
 

<h2>🍽🏫 학교를 검색하여 해당 학교의 오늘의 급식 식단을 볼 수 있는 앱</h2>
  
  <h2>❗ 기능</h2>
   
  ✔ 자신이 원하는 학교 검색 가능
  
  ✔ 선택한 학교의 오늘의 급식 확인
  
  ✔ 아침/점심/저녁 급식 모두 확인
  
   
  <h2>❕ 기술</h2>
  
  ✔ Retrofit + okhttp
  

  <h2>🛠 사용툴/언어</h2>

  ◾ Kotlin

  ◾ Android Studio
